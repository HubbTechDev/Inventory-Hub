### Viewing the Changes

To see the design in action:

1. Install dependencies:
   ```bash
   cd mobile
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Run on a device or emulator:
   ```bash
   # iOS
   npm run ios
   
   # Android
   npm run android
   ```
